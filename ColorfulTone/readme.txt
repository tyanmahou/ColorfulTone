ColorfulTone.exeをクリック	…	ゲームの起動

NotesViewer.exeをクリック	…	譜面データのビジュアライザーを起動

■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

Enter		…	決定
Esc		…	戻る(シーン)
BackSpace	…	戻る
方向キー	…	選択

■シーン別説明
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

●タイトル画面
モードを選択して決定
・FREE PLAY	…	通常モードでゲームを開始します
・COURSE PLAY	…	コースモードでゲームを開始します
・CONFIG	…	コンフィグをします
・DOWNLOAD	…	楽曲データ等のダウンロードをします。
・RELOAD	…	楽曲データのリロードをします
・ACCESS	…	カラフルトーンのWebページを開きます。
・EXIT		…	ゲームを終了します

●コンフィグ画面
・表示するクリアレート
加算式　…　0からはじまりミス以外をとると増えていく
減算式　…　100からはじまりパーフェクト以外をとると減っていく
ライフ　…　コースモードのようにライフ計算で見ることができます。
※通常モードの場合ライフが0になっても強制終了はしません
・円切り取り
ONにすると円の範囲内しかノーツが表示されなくなります。
・プレイ画面の拡大率
プレイ画面の拡大率を変更することができます。
縮小することで、ノーツの大きさは小さくなりますがより先の範囲まで見ることができるようになります。
・キーコンフィグ
変更したいキーを決定して、変更先のキーを入力

●選曲画面
・しぼりこみ選択
上下で楽曲のしぼりこみをします
決定キーで決定

・楽曲選択
上下で楽曲を選びます
BackSpaceキーでしぼりこみ選択が可能です
決定キーで決定

・レベル選択
上下でレベルを選択して決定キーで開始
BackSpaceキーで楽曲選択に戻る

・Ctrl+上下キーでハイスピードの変更ができます。
・Shiftキーでミニ表示の譜面データの情報を切り替えします。
・F10キーで楽曲をお気に入りに登録/解除できます。
・F11キーでコンフィグ画面を開けます。
●メイン画面

オレンジ色の判定ラインにノーツの中心が重なった時にタップします。
ロングノーツは終点が通り過ぎるまで押し続けます。

赤ノーツ…赤ボタン(初期Fキーなど)
青ノーツ…青ボタン(初期SPACEキーなど)
黄ノーツ…黄ボタン(初期Jキーなど)
緑ノーツ…青ボタン＋黄ボタン
橙ノーツ…赤ボタン＋黄ボタン
紫ノーツ…赤ボタン＋青ボタン
黒ノーツ…赤ボタン＋青ボタン＋黄ボタン
白ノーツ…ボタンをおさない(押すとミス)
虹色ノーツ…終わるまで連打(8分間隔よりあくとミス)

BackSpace長押しでリザルトへ
Esc長押しで選曲画面へ

80%を超えるとクリアになります


●リザルト画面

・Enterで遷移
・F10キーで楽曲をお気に入りに登録/解除できます。
・Tキーでリザルトのツイートができます。

・グラフについて
青線…ライフの遷移
棒線…総ノーツを20分割した範囲で、高さがレート色は一番悪かった判定を示します。

●コースモード

・ライフ制になる
・パーフェクト以外をとるとダメージをうけてパーフェクトをとると回復
・ライフを残し切り指定された曲を完走すると合格

●ダウンロード画面

・インターネットから楽曲データ等がダウンロードできます。
・Enterでダウンロード。（既にダウンロード済みのものは文字が薄くなります）
・ダウンロード後はリロードの必要があります
